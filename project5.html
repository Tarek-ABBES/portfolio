Controlling a collaborative robot using Phantom Omni with ROS2 Humble

Project Description

This project involves using a Phantom Omni haptic device to precisely control a collaborative robot, with all communication and control managed by ROS2 Humble. A real-time teleoperation, using moveit 2 servo.

Tools:

Ros2 Humble
C++
OOP
Moveit2 servo
